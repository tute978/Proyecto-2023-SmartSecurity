    ¿Qué es Smart Security?
    Smart Security es un un sistema de control de accesos para una casa mediante tarjetas, huella digital, código y llave. Contiene una cámara para poder ver quien está en la puerta, ademas de una base de datos en la cual se almacena un registro del momento en el qué se abre la puerta. También tiene un sistema qué envia un mail cada vez qué se intenta forzar la cerradura.
